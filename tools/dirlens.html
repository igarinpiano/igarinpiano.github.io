Features

「ls と grep の往復」を、終わらせる。

Dirlens は tree 互換の表示ツールであると同時に、AI・コーディングエージェントがプロジェクトを一度で把握するための解析エンジンです。

🌳
tree 互換 + α
-a -d -f -L -P -I --prune など主要フラグが tree と互換。gitignore 除外は git check-ignore による厳密判定(2層構成で git 不在時は内蔵マッチャに自動縮退)。
🔢
BPE 正確トークン数
o200k_base による正確なトークン計数をバイナリに同梱。言語別内訳・永続キャッシュ付き。「このリポジトリ、コンテキストに入る?」が一目で分かる。
🧠
--agent 一発解析
トークン数・最終コミット・TODO・テスト未整備・エントリーポイント・アウトライン・import 依存グラフ・設定ファイル一覧を一括表示。カラーなし・クリップボード不使用で自律実行にも安全。
🌲
12言語 AST アウトライン
Python / JS・TS / Rust / Go / C / Java / Ruby / PHP / C# / Kotlin / Swift の AST パーサで関数・クラスを正確に抽出。構文エラー時は正規表現へ自動縮退し、どの層で取得したかも機械可読。
🕸️
import 依存グラフ
tsconfig paths・go.mod・Rust モジュールツリー等のマニフェストまで読んで解決。--focus FILE で「このファイルを変えたら何が壊れうるか」を推移閉包で即答。Mermaid / DOT 出力も。
🎯
トークン予算
--budget 3000 で出力自体を指定トークン数以内に自動調整。自前の BPE で実測しながら深さ→詳細→ツリー行の順に削減。--estimate で事前見積もりも。
📡
差分だけを取る
--since HEAD で前回コミット以降の変更ファイルだけのツリー。git diff --name-only | dirlens --stdin でパイプ連携。--api-diff で公開 API の破壊的変更も検出。
🧑‍💻
人間にも、ちゃんと便利
インタラクティブ TUI(-i)・git status オーバーレイ・ヒートカラー・重複検出(--dupes)・ディレクトリ比較(--compare)・HTML レポートまで搭載。
🛡️
正直な解析
解析は「最良の方式 → 縮退」の多層構成。いまどの方式が使われているかは --check で機械的に確認でき、JSON 出力は部分的な失敗があっても常に valid。

MCP Server

エージェントの、ネイティブツールに。

dirlens --mcp で MCP サーバーとして起動。Claude Code・Claude Desktop・Cursor から9つのツールをシェル往復なしで呼べます。トークン予算・コスト見積もり・出力上限対策まで組み込み済み。

analyze tree outline imports focus todos since history api_diff
$ claude mcp add dirlens -s user -- dirlens --mcp

設定に迷ったら dirlens --mcp-setup。ホスト別の登録手順を、バイナリの絶対パス入りで出力します。コピペで完了。

Install

30秒で、手元に。

機種別のネイティブバイナリ(macOS arm64/x64・Linux arm64/x64・Windows x64)。Python も Node ランタイムも不要です。

$npm install -g dirlens

全 OS 対応。機種別のネイティブバイナリが自動で選択されます。Windows でもネイティブな dirlens.exe が動作します。

$tar -xzf dirlens-*-aarch64-apple-darwin.tar.gz
$sudo install -m 755 dirlens /usr/local/bin/

GitHub Releases からお使いのプラットフォームのアーカイブを取得し、PATH の通った場所へ置くだけ。

$cargo install dirlens

crates.io からソースをコンパイルしてインストールします。

$cargo binstall dirlens

crates.io のメタデータから GitHub Releases のビルド済みバイナリを取得。コンパイル不要で cargo install より高速です。
